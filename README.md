# MarIAnada
Pela extensa poluição e mudanças climáticas, os recifes estão morrendo e, com eles, muito da vida marinha e terrrestre também.
Então, surgiu a MarIAnada,um aplicativo que melhora monitoramento da biodiversidade dos litorais brasileiros e ainda por cima aquece a economia local

![image](https://github.com/anacartola/MarIAnada/assets/136506553/152c0ad5-7bca-4417-a188-43ebc24d7046)

A ideia é simples: unir lugares litorâneos de alto fluxo turístico no Brasil, aquecer a economia com realidade aumentada e promover monitoramento de recifes para pesquisas e mais.

![image](https://github.com/anacartola/MarIAnada/assets/136506553/3493d8ef-9b8e-42aa-b91c-5d58191f890f)

Utilizando uma arquitetura Lambda no Data Lake, considera-se que a estrutura auxiliará a integrar todas as etapas, desde o envio das imagens, conversão de pontos em cupons e até o desenvolvimento de relatórios.

Os modelos de ML e operação de IA separados dos bancos permite que ajustes finos possam ser feitos sem o receio de prejudicar o fluxo de ETL. O uso de Azure integralmente foi para facilitar a natividade deste projeto, mas é possível migrá-lo para outra cloud sem problemas.
